Объявление CSS стиля

В head пишем style в <> открывающий и закрывающий

Внутри style пишем ".имя {}" объявляющий css класс, так же добавляем имя в <> стилизируемой таблицы с помощью class:"" или id:""

Настройки внутри CSS стиля

Выравнивание текста text-align:

Имеет 3 значения: left, center, right

Цвет текста color:

Мы можем указать просто цвет на английском или html код цвета

Размер шрифта font-size

Указываем в пикселях

Обводка границы блока border

border: (размер в px) (какая линия) (цвет)

Отступ от элементов margin

Отступ вниз margin-bottom px

Отступ вверх margin-top px

Начертание шрифта font-style

font-style: normal Обычное начертание текста

font-style: italic Курсивное начертание.

font-style: oblique Наклонное начертание Курсив и наклонный шрифт при всей их похожести не одно и то же. Курсив это специальный шрифт имитирующий рукописный, наклонный же образуется путем наклона обычных знаков вправо.

font-style: inherit Наследует значение родителя
